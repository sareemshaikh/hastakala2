# Hasta-Kala Shop

## Overview
Hasta-Kala Shop is an Android-based inventory and sales management application developed for artisans and handicraft sellers. The application helps manage products, inventory, sales records, stock monitoring, and analytics.

## Problem Statement
Small artisans and handicraft sellers often manage inventory and sales manually, leading to stock mismanagement and difficulty in tracking revenue. This project provides a digital Android solution for inventory and sales management.

## Features
- Product Management
- Inventory Tracking
- Sales Record Management
- Dashboard Analytics
- Firebase Integration
- Room Database Support
- MPAndroidChart Analytics Visualization

## Technologies Used
- Kotlin
- Android Studio
- Firebase
- Firestore
- Room Database
- MVVM Architecture
- Material Design
- MPAndroidChart

## Project Structure
- app/src/main/java
- app/src/main/res
- Firebase Configuration
- Room Database
- ViewModel and Repository Layers

## Installation Steps
1. Clone the repository
2. Open in Android Studio
3. Sync Gradle
4. Add Firebase configuration if required
5. Run the application on emulator or Android device
